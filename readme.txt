== Description ==

Popular post comes with some interactive features. Polularity tracking can be handle from both frontend and backend post page.

<strong>[Popular Posts](http://www.giribaz.com/)</strong> with tons of features and priority support.
<strong>[Documentation](http://www.giribaz.com/popular-posts/)</strong> See detail documentation here.

= Key Features =
<ul>
   <li><strong>Popular Post :</strong> Arrange popular posts order by its hit popularity.</li>
   <li><strong>Awesome :</strong> Awesome UI layout to view populas posts.</li>
   <li><strong>Widget Support :</strong> Interactive widget option for plugin tuning.</li>
   <li><strong>Easy Installation :</strong> Easy Instalation for all kind of users.</li>
   <li><strong>Hit Counter :</strong> Post hit counter to show the total number of hits on that post.</li>
   <li><strong>Thumbnail View :</strong> Popular post summery in card view with thumbnail attachment.</li>
   <li><strong>Page Builder :</strong> Works fine with various page builder.</li>
</ul>

= Requirements =

1. At least Firefox 12, Internet Explorer 9, Safari 6, Opera 12 or Chrome/Chromium 19
2. PHP 5.2+ (for thumbnails - mogrify utility or GD/Imagick module)
 
== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Goto the any of the post and check the Custom Fields from Screen Options placed above.
4. Tune your plugin in your way using GB Ppopular Post widget.
5. Enjoy !
